CGPA Mate is a connected CGPA planning workspace built with Next.js (App Router) and Tailwind CSS. It lets students estimate subject results, roll them into semester CGPA, and track total CGPA with smart suggestions.

## Getting Started

Run the development server:

```bash
npm run dev
```

Then open http://localhost:3000.

## Features

- Live subject result estimator with grade and GPA mapping
- Semester CGPA calculator linked to saved subjects
- Total CGPA dashboard with target planning
- Local persistence using localStorage
- Light and dark themes
